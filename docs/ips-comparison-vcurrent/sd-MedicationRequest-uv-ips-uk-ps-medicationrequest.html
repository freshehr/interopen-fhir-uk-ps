    descriptionThis profile represents the constraints applied to the MedicationRequest resource by the International Patient Summary (IPS) FHIR Implementation Guide, based on FHIR R4. A record of a medication request is represented in the patient summary as an instance of a MedicationRequest resource constrained by this profile.This profile sets minimum expectations for a MedicationRequest resource in the context of a patient summary in a UK context. It is based on the [UKCore MedicationRequest](https://fhir.hl7.org.uk/StructureDefinition/UKCore-MedicationRequest), and applies the additional requirements from [MedicationRequest (IPS)](http://hl7.org/fhir/uv/ips/StructureDefinition/MedicationRequest-uv-ips).
